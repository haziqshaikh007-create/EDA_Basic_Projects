# Exploratory Data Analysis (EDA) Projects

This repository contains Exploratory Data Analysis (EDA) projects on **Student Performance** and **Movie Ratings**, aimed at uncovering meaningful insights from real-world datasets using Python. The analysis is performed using libraries such as **Pandas, NumPy, Matplotlib, and Seaborn** to clean, process, and visualize data effectively. In the Student Performance project, factors influencing academic results such as study time, parental education, and other demographics are explored, while the Movie Ratings project focuses on understanding trends in audience preferences, rating distributions, and relationships between genres and ratings. These projects demonstrate key data science skills including data cleaning, handling missing values, statistical analysis, and data visualization to derive actionable insights.
