# Exploratory Data Analysis (EDA) Projects

This repository contains Exploratory Data Analysis (EDA) projects on **Student Performance**, **Movie Ratings**, and **Workout Data**, aimed at uncovering meaningful insights from real-world datasets using Python. The analyses are performed using libraries such as **Pandas, NumPy, Matplotlib, and Seaborn** to clean, process, analyze, and visualize data effectively. In the Student Performance project, factors influencing academic results such as study time, parental education, and other demographic variables are explored. The Movie Ratings project focuses on understanding audience preferences, rating distributions, and relationships between movie characteristics and ratings. The Workout Data project examines workout duration, pulse rate, maximum pulse, calorie expenditure, and workout trends over time to identify patterns in exercise performance and intensity. Together, these projects demonstrate essential data science skills including data cleaning, handling missing values, exploratory analysis, statistical investigation, feature interpretation, and data visualization to derive meaningful insights from diverse datasets.
